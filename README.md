# COMP333 (SWE) semester long project

## Overview of the project
This project is a library booking web application that is designed to enhance how students, faculty, and staff reserve study spaces on the [Wesleyan University](https://www.wesleyan.edu) campus. We currently support study spaces in the following locations: [Olin Library](https://share.google/oIaVXJvfKmKUvlS6M), [Science Library](https://share.google/e8WiViN1jltIHE4Gc).

## Features
The website allows users to view real-time room availability, filter rooms by capacity or location, and securely reserve spaces through an intuitive calendar interface.
